Как установить sqlite на windows

Установка sqlite3 для виндовс:
шаг 0:
скачайте заголовочные файлы из сорца
а также Precompiled Binaries for Windows
по ссылке https://www.sqlite.org/download.html

Для того чтобы всё это установить в ваш MinGW нужно:

шаг 1:
создайте линкующююся библу libsqlite3.a из sqlite3.def с помощью dlltool.exe из MinGW:
dlltool.exe -d sqlite3.def -l libsqlite3.a

шаг 2:
заголовочные файлы (sqlite3ext.h - обязательно ли? и sqlite3.h) поместите в папку C:/MinGW/includes
sqlite3.dll поместите в MinGW/bin (или в папку с проектом)
а libsqlite3.a в MinGW/lib (или в папку с проектом)

Готово!
Теперь вы можете линковаться g++ sqlTest.cpp -o bin/sqlTest -l sqlite3

	Возможные ошибки:
1) !не забывайте что -l указываеться в конце параметров gcc!
https://askubuntu.com/questions/194193/why-do-i-get-undefined-reference-errors-when-linking-against-openssl
А то может возникнуть ошибка gcc link "undefined reference to `symbol`"

2) компилиться, но при выполнении ничего не происходит (printf() не работает)
и при дебаге даже вообще не запускаеться, говорит вышел не запустившись с ошибкой 0xc000007b
возможно вы перепутали 64 и 86 версии dll!
возможно поможет скачивание dll библиотеки другой разрядности
(см. https://stackoverflow.com/questions/11648621/c-sdl-native-has-exited-with-code-1073741701-0xc000007b
в частности инструмент http://www.dependencywalker.com/ )
